# Design System

> Заготовка. Наповнюється в міру побудови UI (модуль «Design system + UI shell»).
> Джерело істини для токенів — Tailwind theme; цей файл документує рішення, не дублює конфіг.

## Правило додавання токенів

Токен додається в `@theme` лише якщо значення **повторюється в кількох місцях** або є частиною дизайн-системи (колір, радіус, тінь, тривалість). Одноразові значення — arbitrary value прямо в класі (`rounded-[6px]`, `shadow-[...]`). `:root` — тільки для значень що змінюються через JS або `@media`.

Назва токена описує **призначення**, не елемент: `--shadow-card` ✅, `--shadow-sidebar` ❌.

## Дизайн-токени

Джерело значень — `src/index.css` (`@theme`). Тут — призначення кожного рівня.

### Кольори — поверхні

| Токен       | Призначення                                    |
| ----------- | ---------------------------------------------- |
| `bg`        | Фон сторінки                                   |
| `bg-subtle` | Зони з легким контрастом (рядки таблиці, теги) |
| `surface`   | Картки, панелі, сайдбар                        |
| `border`    | Розділювачі, обводки елементів                 |

Accent (sage) — `accent` / `accent-subtle`. Небезпечні дії, успіх, помилки — _TBD_.

### Радіус

Три рівні: `sm` → елементи (badge, tag), `md` → поля вводу, кнопки, `lg` → картки, панелі, сайдбар.

### Тіні

`shadow-card` — для піднятих блоків: сайдбар, картка, дропдаун.

### Transitions

`duration-fast` — hover/focus, collapse. `duration-normal` — модалі, sheets.

### Spacing

Стандартна Tailwind scale (4px base). Відхилення документувати тут якщо з'являться.
